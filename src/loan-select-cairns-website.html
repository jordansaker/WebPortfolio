Loan Select Cairns Website

Posted: 14 Jun 2023 | By: Jordan Saker

This project was a collaboration project with a student from Coder Academy, Joshua Phillips. The entire Loan Select Carins website was redesigned using predominantly HTML and CSS, with some Javascript. The intent of the project was to use the need for the Loan Select Cairns website to be updated to a more modern design as project to experiment and use what we've learned with HTML and CSS.

The key goals for the project were:

  • Complete responsive design for mobile, tablet, and desktop devices.
  • Allow news articles to be posted to the website without the need to update the static HTML files.
  • Implement a contact form using the EmailJS service.

Loan Select Cairns News Page design involved using Wordpress as the content management system and accessing the Wordpress API. Through Javascript, get requests were sent to the Wordpress API to fetch post content from a created Loan Select Cairns Wordpress site. The content was then dynamically added to the static HTML file to display the news article posts.
