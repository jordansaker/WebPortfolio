Harvard Reference Generator Markdown

Posted: 1 Jun 2023 | By: Jordan Saker

This project was an assessment project for Coder Academy. The objective of the project was to create an application that would run within the terminal. I chose to implement a basic application that would allow the user to record and insert their references into a Markdown file.

For the project, the main requirements were:

  • The application had to be created using Python.
  • No GUI implementation, meaning the app had to be used entirely in the terminal.
